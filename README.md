面向教育领域的智能问答平台，融合基于知识库的精确检索与基于大模型的语义理解，为学习者提供多学科、多轮对话的专业答疑服务。系统采用「BM25 关键词检索 + RAG 增强生成」双通路架构，覆盖多学科领域知识，支持流式交互与对话历史管理。
技术栈：RAG +  Milvus +MySQL + Redis + LangChain + FastAPI + BGE-M3 + BERT + DashScope + RAGAS

主要工作：

设计并实现了 BM25 + Redis 缓存的知识库精确检索模块，通过 Softmax 置信度阈值策略实现关键词匹配与语义理解的自动分流
开发了支持 PDF/Word/PPT/图片的多模态文档加载与 OCR 识别模块，以及面向中文文本的自定义递归分割器
搭建了基于 Milvus 向量数据库的 RAG 检索增强生成流程，集成 BGE-M3 混合向量检索与 BGE-Reranker 重排序，并设计了直接检索、问题回溯、子查询分解、HyDE 四种检索策略，由 LLM 自适应选择
使用 BERT-base-chinese 微调实现了查询意图分类器，对 5000 条标注数据进行训练，区分"通用知识"与"专业咨询"两类查询并路由至不同处理分支
使用 RAGAS 框架对系统进行多维度评估，忠实度 0.95、答案相关性 0.92、上下文相关性 0.90、上下文召回率 0.93，验证了多策略检索与重排序方案对生成质量的提升效果
